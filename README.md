# EpiDump

**This docker runs on AMD64 architecture !**

**On Apple chips this docker simulated AMD64 architecture !**

```bash
sudo docker run --rm -it -v $PWD/:/test linkheroes/epidump
```

**This container open directly test folder when begin**
